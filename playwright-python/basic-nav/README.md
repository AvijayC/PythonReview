# Playwright Python: Basic Navigation

This lesson directory is intentionally code-light.

The environment setup is ready, but there is no starter solution checked in. Future exercises here should be written manually from simple cases to more complex cases.

## Lesson Goals

- understand what Playwright installs for Python
- learn the role of `sync_playwright()` and the browser/page objects
- write navigation examples from scratch
- later grow into selectors, assertions, and pytest integration

## Environment Status

The repo virtual environment has Playwright installed, and Chromium has been downloaded for Playwright use.

## Working Rule For This Lesson

- do not keep prewritten solution code in this directory before the lesson step needs it
- add examples only when they are being taught or practiced
- keep the `autolessons` log as the continuity record
